# AIRegulate
AIRegulate analyzes AI policy documents using a RAG pipeline with LangChain, Pinecone, and GPT-4o-mini. Features input/output guardrails, anti-hallucination techniques, and evaluation framework.

Usage: Ask questions about AI regulations across China, EU, and US. Get grounded, source-cited responses.
